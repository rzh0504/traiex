## Full Description (English)

traiex is a clean and customizable new tab extension for Microsoft Edge. It replaces the default new tab page with a focused start page that keeps common actions close at hand without adding clutter.

Use the built-in search box with Google, Bing, DuckDuckGo, or Baidu. Manage quick links in the Dock for frequently used websites. Organize bookmarks into custom categories, reorder items with drag and drop, and use long-press sorting directly on the new tab page on both desktop and touch devices.

traiex also includes light and dark themes, background color customization, bilingual Chinese and English interface support, optional local search history, bookmark and history suggestions while typing, and import/export for backup and restore.

The extension only requests the `storage` permission. Settings are stored in browser storage, and optional search history stays on the local device. There is no traiex account system and no traiex backend service.

## Full Description (Chinese)

traiex 是一个简洁、可定制的 Microsoft Edge 新标签页扩展。它会替换默认新标签页，提供更专注、更清爽的起始页面，让常用功能更容易触达，同时避免界面臃肿。

你可以使用内置搜索框，并在 Google、Bing、DuckDuckGo、百度之间切换搜索引擎；也可以通过 Dock 管理常用站点快捷链接。书签支持自定义分类、拖拽排序、跨分类移动，并且在桌面端和触屏设备上都支持在新标签页中长按后直接排序。

traiex 还提供浅色和深色主题、背景颜色自定义、中英文双语界面、可选本地搜索历史、输入时的书签和历史联想，以及导入导出功能，方便备份和恢复全部配置。

扩展目前仅申请 `storage` 权限。设置保存在浏览器存储中，可选搜索历史仅保存在本地设备。traiex 不提供自建账号系统，也不依赖 traiex 自有后端服务。
