# Store Description

## English

### Short Description

traiex is a clean and customizable new tab page for Microsoft Edge with search, quick links, bookmarks, and theme options.

### Full Description

traiex is a clean, focused, and customizable new tab extension for Microsoft Edge.

It replaces the default new tab page with a lightweight start page that helps you get to common actions faster. You can search the web, manage quick links, organize bookmarks, and adjust the page to match your workflow without adding unnecessary clutter.

Key features:

- Custom new tab page for Microsoft Edge
- Search with Google, Bing, DuckDuckGo, or Baidu
- Quick access Dock for frequently used websites
- Bookmark categories with drag-and-drop sorting
- Light and dark themes with background customization
- Chinese and English interface support
- Optional local search history and live suggestions
- Import and export for backup and restore

traiex only uses the `storage` permission to save settings and local data inside the browser. It does not require an account, and it does not run a dedicated backend service.

## 中文

### 简短描述

traiex 是一个简洁且可定制的 Microsoft Edge 新标签页扩展，提供搜索、快捷链接、书签管理和主题设置。

### 详细描述

traiex 是一个简洁、专注、可定制的 Microsoft Edge 新标签页扩展。

它会替换默认新标签页，提供一个轻量、清爽的起始页面，让你更快完成常用操作。你可以直接搜索网页、管理常用站点快捷链接、整理书签分类，并根据自己的使用习惯调整页面外观，而不会引入多余的界面负担。

主要功能包括：

- 自定义 Microsoft Edge 新标签页
- 支持 Google、Bing、DuckDuckGo、百度搜索
- 提供常用网站快捷访问 Dock
- 支持书签分类和拖拽排序
- 支持浅色、深色主题和背景自定义
- 支持中文和英文界面
- 支持可选本地搜索历史和实时联想
- 支持导入导出，便于备份和恢复

traiex 仅使用 `storage` 权限，在浏览器内保存设置和本地数据。不需要注册账号，也不依赖独立后端服务。
