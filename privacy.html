English

Privacy details

What traiex does

traiex is a browser new tab extension. It shows time and date, provides a search box that uses Chrome's default search provider, lets you manage quick links and bookmarks, and stores your preferences in browser storage.

Data stored by the extension

  • Settings, dock items, and bookmark categories are stored with chrome.storage.sync so they can sync with your signed-in browser profile.
  • The extension does not create an account, run its own backend, or upload your data to a traiex server.

When data is sent to third parties

  • When you submit a search, the text you enter is sent through the Chrome Search API to your current default search provider in Chrome.
  • When bookmark favicons are displayed, the bookmark domain is sent to DuckDuckGo's favicon service to request the icon image.
  • External websites that you open from quick links, bookmarks, or search results receive the normal browser request for that page.

Permissions

  • storage is used to save settings, dock items, and bookmark categories in browser storage.
  • search is used only to submit the user's typed query from the new tab search box through Chrome's Search API, so the extension respects the user's current default search provider.

Data sharing and selling

traiex does not sell your personal data. It does not share your data with data brokers. Data is only sent to third-party services when the feature requires it, as described above.

Your choices

  • You can remove synced settings by resetting the extension or uninstalling it from your browser.

中文

隐私说明

traiex 会做什么

traiex 是一个浏览器新标签页扩展。它会显示时间和日期，提供一个使用 Chrome 当前默认搜索引擎的搜索框，支持管理快捷链接和书签，并把你的偏好设置保存在浏览器存储中。

扩展会存储哪些数据

  • 设置项、Dock 站点和书签分类会保存在 chrome.storage.sync 中，以便跟随已登录的浏览器账号同步。
  • 扩展不会创建 traiex 自己的账号体系，也不会把这些数据上传到 traiex 自建服务器。

什么时候会向第三方发送数据

  • 当你提交搜索时，你输入的搜索词会通过 Chrome Search API 发送给 Chrome 当前默认搜索提供商。
  • 当页面显示书签 favicon 时，书签域名会发送到 DuckDuckGo 的 favicon 服务，用于请求图标图片。
  • 当你从 Dock、书签或搜索结果打开外部网站时，目标网站会收到正常的浏览器访问请求。

扩展权限

  • storage 权限用于保存设置、Dock 站点和书签分类。
  • search 权限仅用于从新标签页搜索框提交用户主动输入的查询，并通过 Chrome Search API 遵循用户当前默认搜索设置。

数据共享与出售

traiex 不会出售你的个人数据，也不会把你的数据分享给数据经纪商。只有在功能本身需要时，相关数据才会发送给上面列出的第三方服务。

你的可控项

  • 你可以通过重置扩展设置或卸载扩展来移除相关数据。
